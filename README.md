<h2 id="rodando">Rodando a aplicação 🎲</h2>
 
```bash
# Clone o repositorio
$ git clone https://github.com/HugoCortezL/pipefy-clone.git
 
# Acesse a pasta
$ cd pipefy-clone
 
# Instale as dependencias
$ npm install
 
#Inicie o serviço
$ npm run dev
```
